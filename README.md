# FX Risk Hedging & Simulation Project

## Overview
This project analyses foreign exchange (FX) risk and demonstrates different hedging strategies using real market data. It focuses on currency exposure and compares hedging techniques such as futures and options.

## Key Features
- Real-time FX data using yfinance
- Calculation of returns, volatility, and correlation
- FX price normalization and visualization
- Hedging strategy comparison:
  - Unhedged exposure
  - Futures hedge
  - Options hedge
- Monte Carlo simulation for FX rate forecasting

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- yfinance

## Project Structure
- `fx_risk_hedging_project.py` → Main analysis and simulation

## Key Insights
- Demonstrates how FX fluctuations impact international payments
- Shows effectiveness of hedging strategies
- Uses Monte Carlo simulation for risk estimation
